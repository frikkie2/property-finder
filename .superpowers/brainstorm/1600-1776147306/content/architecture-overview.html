Property Finder — Architecture Overview

AI-assisted property identification tool for estate agents

How It Works — The Pipeline

1️⃣

INPUT

Agent pastes a Property24 listing URL

→ Fetches listing page
→ Extracts all photos
→ Extracts price, beds, area, description
2️⃣

AI ANALYSIS

Claude Vision examines every photo

→ House number visible?
→ Roof type, colour, shape
→ Boundary walls, gates, driveway
→ Pool shape & position
→ Trees, garden features
→ Exterior finish (brick, plaster, etc)
→ Street clues (signs, landmarks)
3️⃣

SATELLITE SCAN

Match features against aerial imagery

→ Roof outline matching
→ Pool detection from aerial view
→ Driveway / boundary patterns
→ Plot size estimation
→ Narrow to candidate properties
4️⃣

STREET VIEW VERIFY

Compare front-of-house with candidates

→ Fetch Street View for each candidate
→ AI compares facade, walls, gates
→ Confidence score per match
→ Rank top 3-5 candidates
5️⃣

RESULTS

Agent reviews matches & confirms

→ Top candidates with confidence %
→ Side-by-side photo comparison
→ Map showing location
→ One-click "Confirm" to save
→ Drive-by route suggestion

Tech Stack

Frontend

Next.js web app — simple, clean UI. Works on laptop & mobile browser.

AI Engine

Claude API (Vision) — analyses listing photos & compares with Street View/satellite.

Maps & Imagery

Google Maps Platform — Street View Static API, Maps Static API (satellite), Geocoding.

Storage & Hosting

SQLite (local DB) + Vercel or local Node server. Minimal cost.

Estimated Monthly Cost (5-10 listings/week)

Google Maps APIs ~R100-200
Claude API (Vision) ~R100-200
Hosting (Vercel free tier or local) R0
Total ~R200-400/month
