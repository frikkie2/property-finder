Property Finder — Additional Features & Ideas

Beyond the core — what makes this tool indispensable

MVP (Build First)

📋 Paste URL & Identify

Core pipeline: paste link → AI extracts features → satellite scan → street view verify → ranked results. The bread and butter.

📸 Manual Photo Upload

Fallback for when URL fetch fails — upload screenshots or photos taken at show days. Drag & drop, multiple files.

🗺️ Results + Map View

Candidates plotted on a map with confidence scores. Side-by-side listing vs Street View comparison. "Open in Google Maps" button.

💾 Search History

All past searches saved with results. Re-open any search. Track which properties were confirmed or rejected.

Phase 2 — Smart Features (Build After MVP Works)

🧠 Suburb Intelligence Cache

Pre-scan and index all 12 suburbs once. Build a database: every property's roof type, pool presence, boundary style, driveway. Future searches become near-instant — just query the index instead of scanning tiles live.

⏰ Mandate Expiry Tracker

Sole mandates typically last 3 months. When you identify a property, the tool estimates when the mandate might expire based on listing date. Alerts you: "This property's mandate likely expires in ~6 weeks — add to follow-up list?"

🔔 New Listing Alerts

Monitor Property24 daily for new listings in your 12 suburbs from other agents. Auto-notify via WhatsApp or email: "New listing in Queenswood — 3 bed, R2.4M. Want me to identify it?"

📱 WhatsApp Integration

Share results via WhatsApp (what agents live on). One tap sends: property address, confidence score, Google Maps link, and Street View photo to yourself or a colleague.

🚗 Drive-By Route Planner

Batch multiple candidates from the week into an optimised driving route. "You have 7 unconfirmed properties — here's a 25-minute route to drive past all of them." Opens in Google Maps navigation.

📊 Area Intelligence Dashboard

Over time, the tool builds insight: which suburbs have most activity, average listing prices, which agents are most active in your area. Helps you focus prospecting efforts.

Phase 3 — Power Features (If You Commercialise)

🏘️ Multi-Agency / Team Support

Multiple agents share a suburb intelligence cache. One agent's confirmed match improves results for everyone. User accounts and permissions.

📄 Deeds Office Integration

Once a property is identified, auto-lookup the owner via WinDeed or similar. Show owner name, purchase date, bond holder. (Requires paid API or partnership.)

🔄 PropCtrl / CRM Sync

Push confirmed properties directly into the agent's PropCtrl account as a new lead. No re-typing addresses. Syncs status (identified → visited → contacted → listed).

🌍 Configurable Areas

Any agent can set up their own suburb cluster in any South African city. Pre-scan their area and start hunting. Subscription model per area.

Other Ideas Worth Noting

  • Show day reconnaissance: Agent attends a competitor's open house, takes photos of the street/approach. Upload those photos — the tool uses them as extra clues (street number from a different angle, neighbouring houses visible, etc).
  • Historical Street View: Google keeps old Street View imagery. If the current Street View doesn't match (house was renovated), check older captures — might match the current listing photos pre-renovation.
  • Reverse image search: Use the listing photos in a Google reverse image search — sometimes the same property appears on other portals, blogs, or old listings with the address visible.
  • Municipal records: Cross-reference property features (bedrooms, plot size) with City of Tshwane municipal valuation rolls — available online. Narrows candidates significantly.
  • For-sale board detection: When scanning Street View, look for competitor "For Sale" boards on properties — confirms active listings and their approximate location without needing the full identification pipeline.
  • Neighbour matching: If listing photos show neighbouring properties, match those neighbours via Street View first — they're sometimes easier to identify (unique features, visible numbers) and immediately pinpoint the target property next door.
